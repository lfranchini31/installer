Ubuntu Installer
=

A tool to help with quickly setting up Ubuntu.

You can view a demo here: http://scottrobertson.github.io/installer/

Options
-
- Ubuntu Versions:
  - 12.04
  - 12.10
  - 13.04 (coming soon)
- Web Server
  - Apache
  - Nginx
- Server Software
  - PHP
    - APC
    - MySQL
    - Memcache
    - MongoDB
    - GD
    - Curl
  - Python
    - virtualenv
  - Mysql
  - Memcached
  - MongoDB
- Version Control
  - Git
  - SVN
  - Mercurial/HG
- Miscellaneous
  - Zip
  - Make
  - Curl
  - g++
  - s3cmd
